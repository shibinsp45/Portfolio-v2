# Shibin SP - Personal Portfolio

A sleek, modern, and highly responsive personal portfolio website built with HTML, CSS, and Vanilla JavaScript.

## Features

- **Liquid Glass Aesthetic:** Custom macOS/iOS inspired translucent frosted glass UI elements.
- **Dark/Light Mode:** Seamless toggle between light and dark themes with dynamic color palette adjustments.
- **Fully Responsive:** Perfectly adapts to desktop, tablet, and mobile screens.
- **Custom Animations:** Smooth scroll transitions, marquee effects, and hover micro-interactions.
- **Dynamic Projects Grid:** Beautifully presented project cards in a clean 2x2 grid.

## Technologies Used

- **HTML5:** Semantic markup structure.
- **CSS3:** Custom properties (variables), Flexbox, CSS Grid, media queries, animations, and backdrop filters.
- **JavaScript (Vanilla):** DOM manipulation, theme toggling, and interactive elements.
- **Google Fonts:** `Inter` for clean typography and `La Belle Aurore` for the signature text.
- **Lucide Icons:** Clean and consistent iconography.

## Getting Started

To view the project locally, simply clone the repository and open `index.html` in your favorite web browser. 

```bash
git clone https://github.com/shibinsp45/Portfolio-v2.git
cd Portfolio-v2
```

No build tools or package managers are required.
