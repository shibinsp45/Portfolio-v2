## 📖 About The Project

Portfolio v2 is a complete redesign of my personal developer portfolio, engineered to deliver a premium user experience without relying on heavy frameworks. The design language is heavily inspired by modern, translucent interfaces (like macOS and iOS), utilizing advanced CSS backdrop filters and dynamic lighting to create a "liquid glass" aesthetic.

It is built from the ground up to be lightweight, performant, and deeply interactive, serving as both a showcase of my work and a demonstration of my frontend engineering capabilities.

### ✨ Key Features

* **Liquid Glass UI:** Custom frosted glass navigation and UI components with real-time blur and color saturation.
* **Intelligent Theming:** Seamless Light and Dark mode switching that automatically adjusts typography contrast, glass opacity, and shadow depths.
* **Fluid Responsiveness:** A layout that gracefully scales from large 4K displays down to mobile devices, ensuring zero horizontal scrolling and perfect touch targets.
* **Micro-interactions:** Carefully orchestrated hover states, transform animations, and a continuous scrolling marquee that bring the page to life.
* **Modular Architecture:** Clean, well-commented Vanilla HTML/CSS/JS structure making it easy to maintain and expand.

## 🛠 Built With

This project was built focusing on core web technologies, minimizing dependencies while maximizing performance.

* **HTML5** - Semantic structure and accessibility.
* **CSS3** - Native CSS variables, CSS Grid, Flexbox, and advanced `@media` queries.
* **Vanilla JavaScript** - Lightweight DOM manipulation and theme management.
* **[Google Fonts](https://fonts.google.com/)** - Utilizing `Inter` for unparalleled readability and `La Belle Aurore` for personalized signature branding.
* **[Lucide Icons](https://lucide.dev/)** - Beautiful, consistent, and lightweight SVG iconography.

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

You only need a modern web browser to view this project. No Node.js, npm, or build steps are required.

### Installation

1. Clone the repository
   ```sh
   git clone https://github.com/shibinsp45/Portfolio-v2.git
   ```
2. Navigate into the project directory
   ```sh
   cd Portfolio-v2
   ```
3. Open `index.html` in your browser, or serve it locally using any basic HTTP server (e.g., Python's SimpleHTTPServer, VS Code Live Server).
   ```sh
   # Example using Python 3
   python -m http.server 8080
   ```

## 📱 Responsiveness

The application is thoroughly tested across multiple breakpoints:
- **Desktop (1024px+):** Full 2-column grids and expansive layouts.
- **Tablet (768px - 1024px):** Adjusted paddings and dynamically scaled typography.
- **Mobile (< 768px):** Single-column layouts, touch-friendly navigation, and mobile-optimized glassmorphism.
